# AI Documentation

This folder contains the AI-specific reference docs for `hospital-chatbot-system`.

## Files

- `agent-context.md` defines the project scope, constraints, and safety boundaries for AI-assisted work.
- `skills.md` lists the main AI support areas and the related code or data surfaces.
- `workflow.md` documents the recommended build, authoring, content, and deployment workflows.

## How To Use

1. Read `agent-context.md` before making architecture or chatbot behavior changes.
2. Use `skills.md` to understand which module or content area a change belongs to.
3. Use `workflow.md` when planning implementation order, rollout, or content operations.
4. Use `../../deploy.md` for the practical release runbook.
